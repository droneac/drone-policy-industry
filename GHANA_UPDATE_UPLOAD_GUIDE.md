# Ghana Resource Update Upload Guide

Upload these paths into the existing `droneac/drone-policy-industry` repository
on the `main` branch:

- `app/page.tsx`
- `public/resources/ghana/`

After the files are committed to `main`, the existing `Deploy GitHub Pages`
workflow should run automatically and publish the updated site.

The Ghana source notebook and Ghana classification/resource sections will then
include:

- Ghana RPAS Education and Qualification Framework (KR PDF/DOCX)
- Ghana RPAS Digital Assessment Framework Proposal (EN PDF/DOCX)
- Ghana RPAS Practical Test Auto-Scoring Infographic (KR PNG)
